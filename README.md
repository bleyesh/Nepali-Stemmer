# Nepali Stemmer

## How to Run

### Interactive Mode
```bash
python nepali_stemmer.py
```

### Benchmark Testing
```bash
python benchmark.py
```
## Directory structure

```
├── answer_root.txt
|   (contains the correct root word)
|
├── answer_suffix.txt
|   (contains the correct suffix)
|
├── benchmark.py
|   (benchmarking script)
|
├── input.txt
|   (contains the text corpus)
|
├── nepali_stemmer.py
|   (stemmer code)
|
├── output_root.txt
|   (contains the output root word)
|
├── output_suffix.txt
|   (contains the output suffix)
|
└── README.md
```
